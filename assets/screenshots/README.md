# Screenshots

This directory contains screenshots of the GhostMetrics Gaming Analytics Dashboard.

## Screenshot Files

Place the following screenshot files in this directory:

1. **dashboard-analytics.png** - Dashboard showing analytics overview with charts
2. **login-page.png** - Login page with Google OAuth authentication
3. **trending-games.png** - Trending games section with game cards
4. **live-streams.png** - Live Twitch streams section

## Image Requirements

- **Format**: PNG or JPG
- **Recommended Size**: 1920x1080 or similar aspect ratio
- **File Size**: Optimize images to keep file sizes reasonable (< 1MB each)
